# tmux
Allows multiple terminal sessions to be accessed simultaneously in a single window.

> Start tmux
```bash
$ tmux
```

| Shortcut | Function |
| --- | --- |
| c | Create Window |
| w | list windows|
|n  |next window|
|p | previous window|
|f | find window|
|,  |name window|
|& | kill window|
